idle 中无法导入pygame模块的快速解决方法

发表于 2019-08-26 分类于 Code 阅读次数：

安装pygame

方式一：网站下载 http://www.pygame.org
方式二：cmd 中输入 ‘’‘pip install pygame‘’‘
安装成功检测方法

cmd中输入 python``import pygame
安装成功此处会显示python/pygame的版本号
示例：
$ pythonPython 3.7.1 (default, Dec 14 2018, 13:28:58)
[Clang 4.0.1 (tags/RELEASE_401/final)] :: Anaconda, Inc. on darwin
Type “help”, “copyright”, “credits” or “license” for more information.

import pygame
pygame 1.9.6
Hello from the pygame community. https://www.pygame.org/contribute.html``

idle中使用pygame

打开idle 编译器 输入 ‘’‘import pygame as pg“
如果可以正常使用的话，表示安装成功，pygame安装到当前python的版本中。

bug修复

如果显示：
Traceback (most recent call last): File "/Users/toughpills/games.py", line 7, in <module> import pygame as pg ModuleNotFoundError: No module named 'pygame

这表明由于系统中有多个python版本，pygame安装至另外一个版本的python中，而不在这里，简单的方法就是找到pygame安装的python路径，引用到此处。
具体执行：
ls -1 /usr/lib/python*/site-packages/pygame/__init__.pyc
找到pygame的安装路径，然后复制路径地址；然后在idle中输入：
import sys
sys.path.append('路径地址')
import pygame
执行命令，显示导入成功：

pygame 1.9.6
Hello from the pygame community. https://www.pygame.org/contribute.html
